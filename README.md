## 로컬 환경에서 리액트 앱을 실행하는 법

### 1. Node.js가 설치되어 있지 않다면, 설치 링크에서 Node.js를 설치합니다.

- Node.js가 설치되어 있는지 확인하는 방법

```
node -v
```

- [Node.js 설치 링크](https://nodejs.org/en/download/)

### 2. Yarn을 설치합니다.

- Yarn 설치 방법

```
npm install -g yarn
```

- Yarn이 잘 설치되었는지 확인하는 방법

```
yarn --version
```

### 3. 의존성 모듈을 설치합니다.

```
yarn install
```

### 4. 리액트 앱을 실행합니다.

```
yarn start
```
